moyashi // url.shortener // Api Document

Overview

Moyashi URL Shortener APIを使用すると長いURLを扱いやすいリンクに変換できます。

HTTPSプロトコルを使用し、JSONフォーマットでデータをやり取りします。

Base Information

Base URL
https://s.moyashi.xyz 必須
Content-Type
application/json 必須

URL Shortening Endpoint

POST /shorten
Request Format 
{
  "url": "https://www.example.com",
  "access_limit": 100,
  "expires_at": "2024-12-31T23:59:59"
}
Response Format 
{
  "shortened_url": "https://s.moyashi.xyz/xxxxx",
  "original_url": "https://example.com",
  "access_limit": 100,
  "expires_at": "2024-12-31T23:59:59",
  "og_data": {
    "title": "Example Domain",
    "description": "Example description",
    "image": "https://example.com/image.jpg"
  }
}

Rate Limits

75
Requests / Minute
Per IP Address
Rate Limit Exceeded
Error
Status Code
HTTP 429
Error Message Too Many Requests
Reset Time
60 seconds

公平な利用とシステムの安定性を確保するため、IPアドレスごとに1分間あたり75リクエストまでに制限しています。

Code Example

Python 
import requests
import json
from datetime import datetime, timedelta

url = "https://www.example.com"
api_url = "https://s.moyashi.xyz/shorten"
headers = {'Content-Type': 'application/json'}

data = {
    'url': url,
    'access_limit': 100,
    'expires_at': (datetime.now() + timedelta(days=7)).isoformat()
}
response = requests.post(api_url, 
    headers=headers, 
    data=json.dumps(data)
)

if response.status_code == 200:
    result = response.json()
    print(f"短縮URL: {result['shortened_url']}")
    print(f"元のURL: {result['original_url']}")
    
    if 'access_limit' in result:
        print(f"アクセス制限: {result['access_limit']}回")
    
    if 'expires_at' in result:
        print(f"有効期限: {result['expires_at']}")
    
    if 'og_data' in result:
        og_data = result['og_data']
        if og_data.get('title'):
            print(f"タイトル: {og_data['title']}")
        if og_data.get('description'):
            print(f"説明: {og_data['description']}")
else:
    print(f"エラー: {response.json()['error']}")

パラメータ
パラメータ 型 必須 説明
url string 必須 短縮したいURL（http://またはhttps://で始まる必要があります）
access_limit integer 任意 URLへのアクセス可能回数を制限します。指定しない場合は無制限となります。
expires_at string 任意 URLの有効期限（ISO 8601形式の日時）。指定しない場合は無期限となります。
